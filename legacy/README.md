# Legacy Code

`legacy/backend/` contains the previous monolithic version of AI Inference Gateway.

It was used during earlier lab stages before the project was decomposed into microservices. The code is kept only for history/reference and to show the evolution of the project.

It is not required to run the final version, and it should not be used for the final Docker Compose or Kubernetes demonstration.

The current final version is located in:

- `frontend/`
- `services/gateway-service/`
- `services/billing-service/`
- `services/task-service/`
- `docker-compose.yml`
- `k8s/`

Use those directories and manifests for the final runtime architecture.
